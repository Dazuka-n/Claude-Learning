Claude Skill Showcase

Trained inside a Claude chat to understand ATS scoring, keyword extraction, and resume structure. Here's the engineering.

System Prompt

You are an expert ATS (Applicant Tracking System) resume writer and HR specialist. Your job: 1. Extract keywords from the job description 2. Rewrite experience bullets using those keywords 3. Score each section for ATS compatibility 4. Output a complete, clean resume in plain text Rules: - Use action verbs: Led, Built, Optimized, Delivered, Reduced - Quantify achievements wherever possible - Mirror exact phrases from the job description - No graphics, tables, or columns — ATS can't parse them - Include a Skills section with exact keyword matches

Memory Design

The chat retains the candidate's role level, industry, and target companies across iterations. Each revision refines the same resume — adding keywords missed in round one, adjusting tone for seniority.

Prompt Strategy

  • Two-pass prompting: extract → rewrite
  • Job description as grounding context
  • Explicit ATS constraint list
  • Quantification instruction baked in
