# SKILL: SINGLE IMAGE POST CREATION
### TTT Social Media Automation | Knowledge Doc
**Use when:** Post type = Single Image

---

## WHAT IS A SINGLE IMAGE POST?

One visual — image, graphic, or designed card — with a caption. It appears as a static post in the feed. On LinkedIn it shows as an image above the caption. On Instagram it appears as a square or portrait photo/graphic.

**Ideal for:** Quotes, quick tips, announcements, stats/data points, event promotions, product highlights, thought leadership one-liners, behind-the-scenes moments.

---

## IMAGE TYPE SELECTION

Before designing, decide which type of image fits the content:

| Content Type | Image Style | Tool |
|---|---|---|
| Quote or insight | Branded quote card | Canva |
| Statistic or data point | Visual stat card with number callout | Canva or Napkin AI |
| Announcement / launch | Bold graphic with headline | Canva |
| Framework or concept | Simple diagram or visual | NanoBanana or Canva |
| Event or milestone | Photo + text overlay | Canva |
| Abstract / AI topic | Illustrated graphic | Canva (illustration templates) |

**Default: Canva for all single image posts**

---

## SIZE SPECIFICATIONS

| Platform | Format | Pixel Size | Notes |
|---|---|---|---|
| LinkedIn | Square | 1200x1200px | Standard feed image |
| LinkedIn | Landscape | 1200x627px | Also works well |
| Instagram | Square | 1080x1080px | Classic |
| Instagram | Portrait | 1080x1350px | Better reach, fills more screen |

**Always use portrait (1080x1350) for Instagram unless content specifically suits square.**
**Use 1200x1200 for LinkedIn as default.**

## STEP-BY-STEP: CANVA SINGLE IMAGE

### Step 1 — Open Canva
Navigate to canva.com. Verify logged in. Click **Create a design → Custom size**.
Enter the correct pixel dimensions for the platform.

### Step 2 — Template Selection
Search for templates using content-specific terms:
- Quote post → search "quote social media"
- Stat/data → search "statistic infographic"
- Announcement → search "announcement modern"
- Tip post → search "tips LinkedIn post"

**Template rules for TTT:**
- Clean and minimal — no clip art, no overly decorative elements
- Brand-appropriate: navy, white, light blue, or dark backgrounds
- Typography-forward — TTT is a thought leadership brand, words matter most
- Avoid templates with obvious stock photo faces unless replacing with real TTT content

### Step 3 — Design the Image

**Layout formula for a TTT branded image:**

```
TOP SECTION (20% of image):
  → TTT logo small top-left OR category label (e.g., "AI Insight" "Quick Tip")

MIDDLE SECTION (60% of image — the hero area):
  → Main headline or quote (large, bold, high contrast)
  → Supporting line (smaller, max 15 words)
  → Optional: icon, illustration, or data callout

BOTTOM SECTION (20% of image):
  → "Teeny Tech Trek" or TTT logo
  → teenytechtrek.com (small)
  → Watermark placement (bottom-right)
```

**Typography rules:**
- Headline: 48–72pt, bold, high contrast against background
- Supporting text: 24–32pt, regular weight
- Never use more than 2 typefaces
- Preferred fonts: Montserrat, Inter, Poppins (all available in Canva)

**Color rules:**
- TTT Primary: #3E6AA7 (use for backgrounds or accents)
- TTT Dark: #1F2A44 (use for text on light backgrounds)
- TTT Accent: #A7C2EA (use for highlights, borders, secondary elements)
- TTT Neutral: #F7F9FC (light background option)
- White text on dark backgrounds, dark text on light backgrounds — never low contrast

### Step 4 — Add TTT Watermark

**Mandatory on every single image post:**

Option A (logo available):
- Upload TTT logo PNG (transparent background)
- Place bottom-right corner
- Size: ~10% of image width
- Opacity: 80%

Option B (no logo file):
- Add text element: "teenytechtrek.com"
- Font: Small, clean sans-serif
- Color: White or #A7C2EA
- Position: Bottom-right corner
- Add semi-transparent dark rectangle behind text for readability if needed

### Step 5 — Review Before Download
Check:
- Text is readable at a glance (3-second test: can you understand it in 3 seconds?)
- No spelling errors
- Watermark is present and visible
- Brand colors used correctly
- Image is not cluttered

### Step 6 — Download
Click **Share → Download → PNG** (not JPG — PNG preserves quality)
Filename: `TTT_[Topic]_[Platform]_[Date].png`
Example: `TTT_AITip_LinkedIn_Apr2026.png`
Save to: Downloads folder

## CAPTION WRITING

**LinkedIn Single Image Caption:**
```
Line 1: Hook — bold statement, question, or surprising fact
[blank line]
Lines 2–4: 2–3 sentences expanding on the visual
[blank line]
CTA: One action — comment, follow, DM, visit site
[blank line]
3–5 hashtags
```

**Instagram Single Image Caption:**
```
Line 1: Hook (makes them tap "more")
[blank line]
Lines 2–5: Story or value expansion
[blank line]
CTA: Save this / share with someone / DM us
[blank line]
10–15 hashtags (mix of sizes)
```

**Caption length:**
- LinkedIn: 150–300 words for thought leadership topics; 50–100 words for tips/quotes
- Instagram: 100–200 words optimal; hashtags separate from body

## PUBLISHING — LINKEDIN IMAGE POST

1. Navigate to linkedin.com/feed
2. Confirm posting from **TTT Company Page** (not personal account)
   - Click **Me → Switch to TTT page** if needed
3. Click **Start a post**
4. Click the **image icon** (🖼) — "Add a photo"
5. Upload the PNG from downloads
6. Add alt text for accessibility: brief description of the image
7. Paste caption in text box
8. Set visibility: **Anyone**
9. Click **Post**
10. Screenshot confirmation

---

## PUBLISHING — INSTAGRAM IMAGE POST

1. Navigate to instagram.com
2. Confirm logged into TTT business account
3. Click **+** → **Post**
4. Upload the PNG
5. Crop if needed (select the correct ratio — square or portrait)
6. Click **Next** → apply filter if desired (usually none for branded graphics) → **Next**
7. Paste caption + hashtags
8. Add location if relevant
9. Click **Share**
10. Screenshot confirmation
