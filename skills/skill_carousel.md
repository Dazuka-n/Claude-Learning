# SKILL: PDF CAROUSEL CREATION
### TTT Social Media Automation | Knowledge Doc
**Use when:** Post type = PDF Carousel / Document Post

---

## WHAT IS A CAROUSEL POST?

A carousel is a multi-slide PDF uploaded as a document to LinkedIn or Instagram. Viewers swipe through the slides. It is the highest-engagement format on LinkedIn. Each slide is a designed visual — not just text. The PDF is uploaded directly to the platform and displayed as a swipeable card.

**Ideal for:** Educational content, step-by-step guides, listicles, case studies, frameworks, tool breakdowns, before/after comparisons.

---

## SLIDE STRUCTURE (Always Follow This)

| Slide | Purpose | Content Rule |
|---|---|---|
| Slide 1 (Cover) | Hook — stop the scroll | Bold headline. One powerful sentence. No clutter. Brand colors. |
| Slide 2 | Problem or Context | Why does this matter? Set up the tension. |
| Slides 3–N | Body / Value | One idea per slide. Short. Visual. Max 40 words per slide. |
| Second-to-last | Key Takeaway | Summarize the core insight in 1–2 lines |
| Last Slide | CTA + Branding | Follow TTT. Visit teenytechtrek.com. TTT logo prominent. |

**Minimum slides:** 5
**Maximum slides:** 12 (LinkedIn optimal: 7–10)
**Words per slide:** Maximum 40. If more is needed, split into two slides.

## STEP-BY-STEP: CANVA CAROUSEL CREATION

### Step 1 — Open Canva
Navigate to canva.com. Verify logged in. Click **Create a design**.

### Step 2 — Select Template
Search for: **"LinkedIn carousel"** or **"Instagram carousel"** depending on platform.

Template selection rules:
- Choose clean, minimal templates — NOT busy or cluttered
- Prefer dark or brand-colored backgrounds for TTT (navy #3E6AA7 or white #FFFFFF)
- Avoid templates with too many stock photos — TTT is text and insight-first
- Pick a template with at least 6 slides

**If no good template found:** Search "presentation minimal" and use a simple slide deck format resized to 1080x1080 (square) or 1080x1350 (portrait).

### Step 3 — Resize if Needed
LinkedIn document posts: **1080x1080px** (square) or **1080x1350px** (portrait — performs better)
Instagram carousels: **1080x1080px** or **1080x1350px**

### Step 4 — Design Each Slide
Work slide by slide. For each slide:
- Clear the placeholder text
- Type the content for that slide
- Keep font size large (minimum 28pt for body, 40pt+ for headlines)
- Use TTT brand colors: Primary #3E6AA7, Dark #1F2A44, Accent #A7C2EA
- One visual element per slide if possible (icon, number callout, simple shape)
- Do not use more than 2 font sizes per slide

### Step 5 — Add TTT Watermark to EVERY Slide
On each slide:
- Add a text element: "teenytechtrek.com" OR upload TTT logo PNG
- Position: Bottom-right corner
- Size: Small — approximately 10% of slide width
- Color: White or light grey
- Opacity: 80%
- If using logo image: upload once, copy-paste to all slides

**Do this for every single slide before downloading. No exceptions.**

### Step 6 — Download as PDF
Click **Share → Download → PDF Print** (highest quality)
File format: PDF
Filename: `TTT_[Topic]_[Date].pdf` (e.g., `TTT_AIAgents_Apr2026.pdf`)
Save location: Downloads folder (or designated TTT output folder)

### Step 7 — Verify the File
Quick check: open the PDF and confirm:
- Correct number of slides
- Watermark visible on all slides
- No placeholder text left
- Last slide has CTA and TTT branding

## STEP-BY-STEP: NAPKIN AI (for data/diagram slides)

Use Napkin for generating visual diagrams from text. Then bring into Canva.

### Step 1 — Open Napkin AI
Navigate to napkin.ai. Verify logged in.

### Step 2 — Input Content
Paste the key insight or data point you want visualized. Napkin will generate visual options.

### Step 3 — Select Best Visual
Choose the clearest diagram. Download as PNG.

### Step 4 — Import into Canva
On the relevant carousel slide in Canva, upload the Napkin PNG as an image element. Resize to fit. Ensure TTT watermark is still visible.

## CAPTION WRITING (for the post itself)

The caption goes in the platform's post text box — not on the slides.

**LinkedIn Caption Formula:**
```
Line 1: Hook (same as slide 1 headline or a teaser)
[blank line]
Line 2-3: What's inside the carousel (2 sentences max)
[blank line]
Line 4: CTA — "Swipe to see all [X] steps →"
[blank line]
Hashtags: 3-5 relevant tags
```

**Instagram Caption Formula:**
```
Line 1: Hook
[blank line]
Lines 2-4: Value summary (what they'll learn)
[blank line]
CTA: "Swipe through →" or "Save this for later"
[blank line]
Hashtags: 10-15 tags (mix of broad and niche)
```

## PUBLISHING — LINKEDIN DOCUMENT POST

1. Navigate to linkedin.com/feed
2. Confirm logged in as TTT Company Page (not personal)
   - Click the **Me** dropdown → Switch to TTT Company Page if needed
3. Click **Start a post**
4. Click the **document icon** (📄) — "Add a document"
5. Upload the PDF from downloads folder
6. Add a title for the document (this shows above the carousel): use the post headline
7. Wait for upload to complete (may take 30–60 seconds)
8. Paste the caption in the text box above
9. Set visibility: **Anyone**
10. Click **Post**
11. Take a screenshot of the published post as confirmation

---

## PUBLISHING — INSTAGRAM DOCUMENT POST

1. Navigate to instagram.com
2. Confirm logged in as TTT business account
3. Click **+** (Create) button
4. Select **Post**
5. Upload all slide images one by one (Instagram doesn't accept PDF — convert slides to PNG first if needed)
   - In Canva: Download as PNG (all pages) — this gives one PNG per slide
   - Upload in correct order
6. Select all images for carousel (up to 10 images)
7. Click Next → Apply filter if needed → Next
8. Paste caption
9. Add location tag if relevant
10. Click **Share**
11. Screenshot confirmation

**Note:** Instagram does not support PDF upload. Always download as PNG images for Instagram carousels.
